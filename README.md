# Dano's Granola Website

A responsive multi-page redesign built as a gift for Dano's Granola.

## Pages
- Home
- Shop
- Our Story
- Visit / Contact

## Publishing on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files and the `assets` folder.
3. Open **Settings → Pages**.
4. Select **Deploy from a branch**, then `main` and `/root`.

Product buttons currently open the existing Dano's online shop so customers can complete purchases through the live store.
